silc_ind_browse

Provide information (report) regarding the definition/data of EU-SILC indicators.

%silc_ind_browse(ind, lib, page=, fm=, odir=, oreport=);

Arguments

  • ind : (list of) indicator(s) whose information is requested;
  • lib : input library where the indicator is stored;
  • page : (option) variable used by proc report to create deparate page for each BY (GEO or TIME are accepted): if not passed GEO is used;
  • odir : (option) pathname where the report is stored, by default: &G_PING_DIRHTML is used;
  • fm : (option) format of values.

Returns

  • oreport : (option) name of the output report; by default: HTML_&ind is used.

Example

The instructions:

%let ind=MDDD01;
%let lib=C_RDB;
%silc_ind_browse(&ind,&lib);
