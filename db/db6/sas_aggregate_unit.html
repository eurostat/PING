aggregate_unit

Set the "unit" field (whose name is either unit or currency) for the aggregate of a given dataset whenever it exists. 

%let flag=%aggregate_unit(dsn);

Arguments

dsn : name of the considered dataset.

Returns

flag : name of the so-called "unit" field returned (whenever it exists for the considered dataset).

Note

The presence of a "unit" field depends on the (name of the) dataset and is "hard"-encoded in the program, namely:

  • currency is returned when tab is in (DI10,DI02),
  • unit is returned when tab is in (DI03,DI04,DI05,DI07,DI08,DI09,DI13,DI13b,DI14,DI14b),
  • nothing is returned (i.e., flag=) otherwise.

This macro program is used for calculating and building the aggregate indicators. See also macro `aggregate_join` and the keyword parameter flag_unit.
