_DSTEST1000

Test dataset #1000. 

%_dstest1000;
%_dstest1000(lib=, _ds_=, verb=no, force=no);

Contents

Arguments

  • lib : (option) output library; default: lib is set to WORK;
  • verb : (option) boolean flag (yes/no) for verbose mode; default: no;
  • force : (option) boolean flag (yes/no) set to force the overwritting of the test dataset whenever it already exists; default: no.

Returns

_ds_ : (option) the full name (library+table) of the dataset _dstest1000.

Example

To create dataset #1000 in the WORKing directory and print it, simply launch: 

%_dstest1000;
%ds_print(_dstest1000);
