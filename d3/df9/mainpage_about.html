Statistical data handling and processing in production environment
About...

PING is a set (library) of macro/function utilities developped in R/SAS for the implementation of statistical processes in production environments.

PING implementation started in April 2016 and it is still ongoing. Its main contributors are:

  • Jacopo Grazzini (gjacopo)
  • Marina Grillo (marinapippi)
  • Pierre Lamarche (pierre-lamarche)
  • Matyas Meszaros (mmatyi)
  • Veronica Nica (Nica87)
  • ...
